# Glide Run Area Review

## Load

- `Glide HQ/AGENTS.md`
- `Glide HQ/Company Context.md`
- `Glide HQ/Marketing Lead Brief.md`
- `Glide HQ/Checklists/Area Review.md`
- `Glide HQ/Areas/AGENTS.md`
- The selected area folder

## Process

1. Resolve the area from `Glide HQ/Areas/`.
2. Load the area's instructions and files.
3. Review objective, context, decisions, questions, reminders, research, and follow-through.
4. Check dependencies across other areas.
5. Recommend updates, next moves, and missing evidence.

## Output

- Area synthesis.
- Risks or contradictions.
- Missing context.
- Suggested next move.
- File updates when useful.
