# Glide Update Marketing Context

## Load

- `Glide HQ/AGENTS.md`
- `Glide HQ/Company Context.md`
- `Glide HQ/Marketing Context.md`
- `Glide HQ/Marketing Lead Brief.md`
- `Glide HQ/Checklists/Update Marketing Context.md`
- Relevant decisions, research, area files, questions, contradictions, and follow-through items

## Process

1. Identify durable marketing, company, or operator signal.
2. Update only context that improves future advice, decisions, research, or operating reviews.
3. Keep stable context concise.
4. Mark uncertainty.
5. Preserve source links when useful.
6. Remove stale recent context when it stops helping.

## Output

- Context updates when durable signal exists.
- A short note when no update is needed.
