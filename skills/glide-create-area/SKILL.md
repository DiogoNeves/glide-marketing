# Glide Create Area

## Load

- `Glide HQ/AGENTS.md`
- `Glide HQ/Areas/AGENTS.md`
- `Glide HQ/Company Context.md`
- `Glide HQ/Marketing Lead Brief.md`

## Process

1. Clarify the area name and purpose.
2. Check whether an existing area already covers it.
3. Create the folder under `Glide HQ/Areas/<Area>/`.
4. Add `AGENTS.md`, `Area.md`, and starter files for context, goals, sources, decisions, questions, reminders, reviews, checklists, and research when useful.
5. Link the area to company context and other areas.

## Output

- Created or updated area.
- Initial purpose and operating notes.
- Suggested next questions.
