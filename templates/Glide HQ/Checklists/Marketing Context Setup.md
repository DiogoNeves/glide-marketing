# Marketing Context Setup

Use when installing Glide Marketing or creating the first useful marketing context.

## Process

1. Inspect available company materials before asking questions.
2. Ask where existing sources and content live.
3. Draft `Source Map.md`, `Content Library.md`, `Company Context.md`, `Marketing Context.md`, and `Marketing Lead Brief.md`.
4. Make a first pass at current pain, goals, and operating process.
5. Ask what is wrong, missing, uncertain, and strategically important right now.
6. Ask what signals need to be checked daily for Glide to be useful.
7. Capture exact customer language when available.
8. Mark uncertain claims as uncertain.
9. Keep the first version useful, not exhaustive.

## Key Sections

- Company overview.
- Source map.
- Existing content.
- Positioning.
- ICP and customer language.
- Funnel.
- Channels.
- Metrics.
- Campaigns and assets.
- Current pain.
- Goals.
- Operating process.
- Active experiments.
- Open questions.
