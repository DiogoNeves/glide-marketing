# Follow-Through Ledger

Lightweight tracking for commitments, waiting threads, promises, and important loops that should not disappear.

This is not a task manager. Use it for follow-through quality.

## Rules

- Track only items that benefit from ongoing attention.
- Treat daily watchlist findings as candidate ledger updates when they create, advance, or close a commitment.
- Keep entries short.
- Link to source context when useful.
- Use external task/project tools for execution when the company uses them.
- Close, simplify, or move stale items when they stop helping.
