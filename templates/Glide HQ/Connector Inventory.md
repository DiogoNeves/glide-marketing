# Connector Inventory

Record the connectors and tools that are actually available in this harness.

Do not list secrets, tokens, private URLs, or credentials.

## Rules

- Discover active connectors during setup from the harness/tool list.
- Do not assume a connector exists because Glide mentions it.
- Reading and fetching are allowed only when the user has granted access.
- External-state changes require explicit approval.
- If connector capabilities are unclear, test with a narrow read-only query or ask the user.
