# Daily Marketing Check-In Prompt

Run the installed `glide-daily-marketing-check-in` skill or follow `Glide HQ/Checklists/Daily Marketing Check-In.md`.

Check `Glide HQ/Daily Watchlist.md` first. Surface one useful marketing question, risk, opportunity, contradiction, access gap, or follow-through nudge.

During the first five daily check-ins, ask one direction question unless the user opts out. Learn current pain, goals, process, missing access, signal quality, timing, tone, length, preferred action level, or what else should be checked daily.

Keep updates brief. When useful, offer one or two paths to explore more. Update `Follow-Through Ledger.md` when daily checks create, advance, or close commitments.
